## init

```sh
$ npm init

```

## install

```sh
# webpack
$ yarn add -D webpack webpack-cli webpack-dev-server webpack-merge html-webpack-plugin clean-webpack-plugin

# typescript
$ yarn add -D typescript ts-loader

```

## config

```sh
$ tsc --init

````
